# Database Map

**Status**: N/A

This project is a static multi-page application consisting of HTML and CSS files.
There is no database connected to this project. All product data, project portfolio information, and copywriting are hardcoded into the respective HTML files in the `/src` directory.

If dynamic database capabilities are added in the future, this document should be updated to reflect the schemas (e.g., Products, Projects, Categories, Enquiries).
